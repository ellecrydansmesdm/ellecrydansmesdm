            <sub><b>🔴 PROJET PRINCIPAL · PRODUCTION</b></sub>
          </td>
        </tr>
      </table>
      <br>
      <p>
        Suite de sécurité locale développée avec <b>Electron + React + TypeScript</b>.<br>
        Audit de vulnérabilités, chiffrement des configs sensibles, surveillance réseau en temps réel — <b>zéro dépendance cloud</b>.

      <ul>

        <li><b>Modules C++ (Node-API)</b> — tâches système critiques hors du thread V8</li>
