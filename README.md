## ⌁ à propos

Je conçois des applications desktop pensées comme des produits, pas comme des scripts.
Chaque interface est construite avec la même exigence : **sombre, épurée, rapide, cohérente**.
Electron est mon terrain de jeu principal — je l'utilise pour transformer des idées techniques
en logiciels que les gens ont réellement envie d'utiliser.
